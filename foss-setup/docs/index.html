How to use this guide

This is a self-contained tracker for migrating off the Apple/iOS ecosystem onto your own hardware. Work top-to-bottom through the five phases — each phase leaves you better off and is safe to stop at. Check a task to mark it done; your progress is saved in this browser automatically (no account, no server). Click any task to expand its grounded steps, copy-paste commands, the config/script files the swarm generated for you, the upstream docs each step is based on, and a concrete "done when" check.

Synchronous — blocking; do in order (a dependency or one-way door) Async — run unattended / in parallel / anytime Host the task runs on

Critical path: UniFi backup → VLANs + Zone-Based Firewall → Tailscale → backups + a tested restore → (Phase 4) Caddy/ntfy → Forgejo → push all configs to Git. Everything else hangs off this spine and much of it parallelizes. The single biggest mistake is treating the firewall migration as async — it's a one-way door, so back up first and do it deliberately.
